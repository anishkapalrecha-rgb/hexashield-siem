# hexashield-siem
Real-time SIEM system with live packet capture, attack detection (NULL/XMAS scans), and interactive packet analysis dashboard.

How to Run HexaShield SIEM (Windows)

##  1. Clone the Repository
bash
git clone https://github.com/YOUR_USERNAME/hexashield-siem.git
cd hexashield-siem


##  2. Create Virtual Environment
bash
python -m venv venv

##  3. Activate Virtual Environment
bash
venv\Scripts\activate

##  4. Install Dependencies
bash
pip install flask scapy

##  5. Run the Application
bash
python app.py

##  6. Open in Browser
Go to:
http://127.0.0.1:5000
